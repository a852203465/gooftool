# ORM(JPA)模块

## 使用方式
### 引入依赖
```xml
<dependencies>
    <dependency>
        <groupId>cn.darkjrong</groupId>
        <artifactId>gooftool-jpa</artifactId>
    </dependency>
</dependencies>
```

### 条件查询
#### 引入依赖
```xml
    <dependency>
        <groupId>com.github.wenhao</groupId>
        <artifactId>jpa-spec</artifactId>
    </dependency>
```

#### 使用
    参考：https://github.com/wenhao/jpa-spec
