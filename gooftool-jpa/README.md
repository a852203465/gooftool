# ORM(JPA) 模块

## 1. 使用方式
### 1.1 引入依赖
```xml
<dependencies>
    <dependency>
        <groupId>cn.darkjrong</groupId>
        <artifactId>gooftool-jpa</artifactId>
        <version>${latestversion}</version>
    </dependency>
</dependencies>
```

### 1.2 条件查询
#### 1.2.1 引入依赖
```xml
<!-- 条件查询封装 -->
    <dependency>
        <groupId>com.github.wenhao</groupId>
        <artifactId>jpa-spec</artifactId>
    </dependency>
```
